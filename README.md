## Features

- **User Authentication:** Sign up, Sign in with JWT-based authentication.
- **Role-Based Access:** Admin and Customer roles with specific permissions.
- **Vehicle Management:** Add, update, delete, and view vehicles (Admin only).
- **Booking System:** Customers can create, view, and cancel bookings; Admin can mark bookings as returned.
- **Data Protection:** Users cannot delete if active bookings exist; vehicles cannot be deleted if currently booked.
- **Responsive API Endpoints:** Fully functional RESTful API endpoints with proper status codes and messages.

## Technology Stack

- **Backend:** Node.js, Express.js, TypeScript
- **Database:** PostgreSQL with node-postgres (pg)
- **Authentication:** JWT (JSON Web Tokens), bcrypt for password hashing
- **Environment Management:** dotenv
- **Version Control:** Git & GitHub
- **Testing Tools:** Postman
